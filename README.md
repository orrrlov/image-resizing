## :wrench: Requirements

- Cloudflare Image Resizing

## :zap: API

- `?w=` width
- `?h=` height
- `?f=` format
- `?q=` quality
- `?bg=` background
- `?dpr=` dpr
- `?fit=` fit
- `?rotate=` rotate
- `?sharpen=` sharpen
- `?gravity=` gravity
- `?metadata=` metadata
